# CodingCamp-20July26-MilaKarmila

## Expense & Budget Visualizer

A responsive web application for tracking daily expenses and visualizing spending by category. This project was built using HTML, CSS, and Vanilla JavaScript as part of the CodingCamp Full Stack Software Engineering assignment.

## Features

- Add and delete transactions
- Input validation
- Total balance calculation
- Spending chart using Chart.js
- Local Storage support
- Dark/Light mode
- Custom categories
- Sort transactions
- Responsive design

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
- Local Storage API
- Font Awesome
